# DodaLock v1.0

## Description

Spigot плагин добавляющий замки и кодовые замки для дверей, люков, бочек и сундуков.

## Features

* Закрывай свои двери, люки, бочки и сундуки от других игроков.
* Возможность использования как обычных, так и кодовых замков.
* Кодовый замок, обычный замок, ключ к нему и связка ключей имеют 
свой крафт.
* Совместим с новыми блоками версии Minecraft 1.19.4 
(даже из вишнёвого дерева).
* Открытый исходный код (лицензия GNU GPL v3).

## How to use it?

Держа в руках кодовый или обычный замок, нажмите в приседе ПКМ 
на дверь, люк, бочку или сундук,
таким образом вы поставите его. 

В случае с кодовым замком, необходимо ввести пароль. Теперь, когда
вы авторизованы, вы сможете открывать этот объект.

В случае с обычным замком необходимо, держа в руках ключ, 
нажать в приседе ПКМ на объект, на котором висит
замок, чтобы создать ключ для данного замка. Теперь, когда вы
имеете ключ от данного объекта, вы можете открывать его, используя
ключ от замка данного объекта.

Чтобы снять какой-либо замок, нажмите в приседе ЛКМ. В случае с
обычным замком, необходимо держать ключ в руках.

### Server Compatibility

Этот плагин совместим с Spigot. 
Использование CraftBukkit не сработает. 
В настоящее время поддерживается версия Minecraft 1.19.4.

### Compilation

DodaLock использует Maven для компиляции и сборки плагина. 
Запустите "maven package" в своей среде IDE, чтобы создать jar-файл. 
Jar-файл плагина будет находиться в папке "target", когда процесс
компиляции закончится.
